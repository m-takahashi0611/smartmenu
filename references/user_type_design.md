## 流入動線

- 全動線共通フロー：友だち追加 → ①トライアル → カード登録 → ②無料期間（20日） → ③プレミアム
- LP・SEO（通常動線）：特典なし
- YouTuber動線：軽い特典（要検討）
- 友達紹介：初回決済X%OFF（管理画面で設定可能）
